# 迷你Wireshark - 网络抓包分析工具

## 🎯 项目简介
一个基于Python开发的图形化网络抓包分析工具，具有类似Wireshark的核心功能，支持协议解析、流量统计、数据可视化和安全检测。

## ✨ 核心功能

### 1. 网络抓包
- 多协议支持：TCP、UDP、ICMP、ARP、DNS、HTTP等
- 实时捕获与显示
- 过滤条件设置
- 网络接口选择

### 2. 数据分析
- 协议分布统计
- IP地址分析
- 流量趋势分析
- Top N统计

### 3. 可视化图表
- 协议分布饼图/柱状图
- 流量时间序列图
- IP地址分析图

### 4. 安全功能
- 数据包匿名化（隐私保护）
- 端口扫描检测
- DDoS攻击检测
- 异常流量分析

### 5. 文件操作
- 数据包保存为JSON格式
- 从文件加载数据
- 导出统计报告

## 🛠 技术栈
- **Python 3.8+**
- **GUI框架**: tkinter
- **网络抓包**: scapy
- **数据可视化**: matplotlib
- **数据处理**: pandas, numpy
